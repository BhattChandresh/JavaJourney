## Programming Language : Java

## Java Version : java version "1.8.0_191"

## Assumption :
- For King,Queen,Rook,Horse and Bishop chess board is always empty.
- For Pawn, only forward direction is considered (A -> H) and it can not move from H -> A.
- Hence we can't place pawn from A1 to A8.
- Becasue the Chess Board is empty, no diagonal movement of Pawn to eliminate the enemy.
